## ğŸ“„ Project Description
This project is a comprehensive web-based student portal for Hogwarts School of Witchcraft and Wizardry. It provides a complete management system for students to register, login, view their academic information, and interact with various school services.

**Purpose:** To create an interactive and visually appealing student management system that demonstrates proficiency in HTML, CSS, JavaScript, and PHP.

**Features Included:**
- Student registration and authentication system
- House-based color theming (Gryffindor, Hufflepuff, Ravenclaw, Slytherin)
- Dashboard with personalized student information
- Academic information (courses, curriculum, grades)
- Staff directory with images
- School policies and administration details
- Contact information
- Notice board
- Class schedules and mess menu
- Academic calendar with PDF download capability
- File-based data storage (no database required)
